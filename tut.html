Hoe hebben wij dit game gemaakt?

Een som spel waar je juiste aantwoord moet geven waar uw auto kan rijzen tussen de verschillende landen.

Nodige codetalen:

  • HTML
  • CSS

Het lijkt misschien moelijk maar het is best wel nog envoudig!
